Calenders

This C++ project facilitates accurate date-related calculations through its modular and class-based structure. Key features include printing date details, checking leap years, validating dates, and displaying detailed calculations. Additionally, the program tracks the usage of the day-of-week calculation service. The design ensures precise operations and a user-friendly experience.

Hotel Booking

This project showcases a responsive and user-friendly frontend design for a hotel booking website. It features a variety of rooms with detailed descriptions and booking options. Users can explore hotel amenities, view contact details, and seamlessly book rooms online. The design emphasizes intuitive navigation and accessibility, enhancing the overall user experience.
